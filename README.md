# Stuff Communication

PWA für zentrale Mitarbeiter-Kommunikation
PhysioPro Lübeck + Pilates Company

## Features
- Admin-Dashboard (Kanban für Nachrichten)
- Mitarbeiter-Dashboard (PIN-Login)
- Lesestatus-Tracking
- Audit-Log
- WhatsApp-Integration (optional)

## Tech Stack
- React
- Google Sheets API
- Netlify Functions
- localStorage für offline
